## instalar node

### Winddows
- https://nodejs.org/en/

### Linux
```sh
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.38.0/install.sh | bash

sudo source ~/.bashrc

nvm install node
```

## Install Serve

```sh
npm install serve -G
```

## Usa inicia el servicio

- Esto es una forma de iniciar
```sh
npx server -l 3000
```
- Esto es otra forma de iniciar el servidor

```
npm run dev
```
